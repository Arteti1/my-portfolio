# Документація портфоліо 

## Опис та принципи роботи
Проєкт реалізований як статичний веб-сайт на базі HTML5 та CSS3. Навігація між розділами (Головна, Контакти, Про нас) здійснюється через стандартні гіперпосилання без використання сторонніх бібліотек чи JavaScript-фреймворків.

## Репозиторії

* **Загальний репозиторій групи:** [kn-2026](https://github.com/PorokhovnichenkoIryna/kn-2026)

## Порядок запуску (Tutorial)
Для локального відтворення проєкту:
1. Виконайте клонування: 
   ```bash
   git clone https://github.com/Arteti1/my-portfolio
2. Перейдіть у папку з проєктом:
   ```bash
    cd portfolio
3. Запуск сайту
Проєкт не потребує спеціального середовища виконання. Просто відкрийте файл index.html у будь-якому браузері (Chrome, Firefox, Edge).
# Результати та набуті навички
У ході виконання цієї практичної роботи було опрацьовано та засвоєно такі теми:

Веб-розробка: Створення семантичної структури сторінок (HTML5) та їх стилізація (CSS3).

Контроль версій (Git): Управління станами проєкту через коміти, синхронізація локальних змін із віддаленим сервером GitHub.

Командна взаємодія: Робота з Pull Requests, призначення Reviewers та процес виправлення зауважень після перевірки коду.

Технічна документація: Робота з мовою розмітки Markdown для створення структурованих описів проєктів.

Вирішення проблем (Debugging): Самостійний пошук та усунення багів, зокрема помилок у локалізації інтерфейсу та навігації.
